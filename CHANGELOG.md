# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial project structure.
- Admin Dashboard with department, course, staff, and student management.
- Faculty Dashboard with course handling and student view.
- Student Dashboard with attendance, marks, and assignment tracking.
- Unified Login Portal with role-based authentication.
- Responsive design for all dashboards.
- Documentation: README, CONTRIBUTING, LICENSE.

### Changed
- Refactored CSS and JS into separate `assets/` directories for better maintainability.
- Updated all HTML files to link to external assets.
- Improved code organization and formatting.

### Security
- Basic role-based access control (frontend simulation).
