# 🎓 Department Management System (DMS)

A **Smart Academic Administration Platform** designed for modern educational institutions. Built with a luxury Apple-level UI, DMS enables seamless management of students, faculty, academics, and attendance through a unified, role-based platform.

---

## 🚀 Features

- 🔐 **Multi-Role Secure Login** — Student, Faculty, HOD & Parent portals with dedicated video backgrounds
- 📊 **Admin & HOD Dashboard** — Full department analytics, department overview, and resource control
- 👨‍🏫 **Faculty Admin Center** — Attendance marking, student evaluation, marks management
- 🎓 **Student Profile View** — Semester-wise marks, CGPA, attendance & academic history
- 📚 **Course Management** — Syllabus tracking, subject assignments, and scheduling
- 🧠 **Smart Campus Intelligence (AIML)** — Predictive analytics, drop-off detection, recommendation engine
- 🌐 **Portfolio Landing Page** — Premium glassmorphism showcase of the platform

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript
- **Design**: SF Pro Display / Inter, Indigo-Violet gradient theme
- **Storage**: LocalStorage (demo), ready for backend integration
- **Fonts**: Google Fonts (Poppins, Inter)
- **Videos**: Dynamic role-based mp4 backgrounds on login

## 🎯 How to Run

1. Clone this repository
2. Open `index.html` in your browser
3. Click **Live Demo** to go to the login page
4. Select a role (Student / Faculty / HOD / Parent) and sign in
